## Task 1 – Flashcard Quiz App

A browser-based **Flashcard Quiz App** that helps users study and test their knowledge through interactive flashcards.

### ✨ Features
- Flip cards to reveal answers
- Navigate between multiple flashcards
- Track score and progress
- Clean and responsive UI

### 📁 Folder
```
task1-flashcard-quiz-app/
```

## Task 2 – Random Quote Generator

A simple and elegant **Random Quote Generator** that displays motivational and inspirational quotes at the click of a button.

### ✨ Features
- Generates a new random quote on button click
- Displays quote author
- Option to share quote on Twitter
- Smooth transitions and animations

### 📁 Folder
```
task2-random-quote-generator/
```

## Task 3 – Fitness Tracker

A personal **Fitness Tracker** web app to log and monitor daily workouts and fitness goals.

- View workout history
- Progress summary and stats
